FastAPI Shop — это backend-приложение интернет-магазина, разработанное с использованием FastAPI. Проект предоставляет REST API для работы с товарами, категориями, пользователями и заказами.

Возможности
Регистрация и авторизация пользователей
Работа с товарами и категориями
Просмотр списка товаров
Создание и управление заказами
